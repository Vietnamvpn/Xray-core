[README](https://github.com/XTLS/Xray-core#readme) là mở, nên bạn có thể gửi dự án của mình [tại đây](https://github.com/XTLS/Xray-core/pulls).

### Phân phối các thành phần bên thứ ba đi kèm

**Một số tính năng tùy chọn nạp động các thành phần bên thứ ba. Những thành phần tùy chọn này là các tác phẩm riêng biệt được phân phối theo giấy phép riêng của chúng, và được đóng gói trong gói ZIP để tiện sử dụng. Người dùng có thể thay thế các thành phần này theo giấy phép tương ứng của chúng.**

Các thành phần này bao gồm:

#### Wintun

Bản phân phối này chứa các tệp nhị phân Wintun chính thức đã được biên dịch sẵn và ký sẵn, không chỉnh sửa.

## Biên dịch một dòng

### Windows (PowerShell)

```powershell
$env:CGO_ENABLED=0
go build -o xray.exe -trimpath -buildvcs=false -ldflags="-s -w -buildid=" -v ./main
```

### Linux / macOS

```bash
CGO_ENABLED=0 go build -o xray -trimpath -buildvcs=false -ldflags="-s -w -buildid=" -v ./main
```

### Phát hành có thể tái tạo

Hãy đảm bảo bạn dùng cùng phiên bản Go, và nhớ đặt id commit git (7 byte):

```bash
CGO_ENABLED=0 go build -o xray -trimpath -buildvcs=false -gcflags="all=-l=4" -ldflags="-X github.com/xtls/xray-core/core.build=REPLACE -s -w -buildid=" -v ./main
```

Đối với Android:

```bash
GOOS=android GOARCH=arm64 CGO_ENABLED=1 CC=/path/to/aarch64-linux-android24-clang go build -o xray -trimpath -buildvcs=false -gcflags="all=-l=4" -ldflags="-X github.com/xtls/xray-core/core.build=REPLACE -s -w -buildid= -checklinkname=0" -v ./main
GOOS=android GOARCH=amd64 CGO_ENABLED=1 CC=/path/to/x86_64-linux-android24-clang go build -o xray -trimpath -buildvcs=false -gcflags="all=-l=4" -ldflags="-X github.com/xtls/xray-core/core.build=REPLACE -s -w -buildid= -checklinkname=0" -v ./main
```

Nếu bạn đang biên dịch mục tiêu MIPS/MIPSLE 32-bit, hãy dùng lệnh sau:

```bash
CGO_ENABLED=0 go build -o xray -trimpath -buildvcs=false -gcflags="-l=4" -ldflags="-X github.com/xtls/xray-core/core.build=REPLACE -s -w -buildid=" -v ./main
```
